# 论文整体检查

同时启用多个子Agent检查论文：
1. 激活 humanizer-zh 检查论文。
2. 激活 thesis-writer 并读取这个 skill 的参考内容，检查论文。
3. 读取 archive 目录下的初稿意见，分析解决情况。
4. 事实检查。多个子Agent检查不同方面的内容。

子Agent只用于分析问题，禁止子Agent修改内容。
这三个子Agent分析完成后，如果发现问题，分析是否可以修复。
如果可以，请在保持内容准确、诚实等符合基本要求的情况下修复。

完成后，使用相同提示词，再次同时启用这些子Agent。如此循环直至问题全部被处理。
注意：
- 不要携带之前的修复内容提示。
- 不要复用已有的会话。

最后给我报告。
