面向城市停车场数据采集的轻量级红外感知无人机集群系统


主要内容：
研发面向城市停车场的红外（可见光）感知无人机集群系统，在无外部定位锚点条件下，依托机载视觉传感器、超宽带（UWB）通信与强化学习调度算法，实现多无人机车位状态数据采集与融合。核心研究内容包括无人机相对定位、车位识别、无人机集群规划三大模块。
计划进度：
1. 前期完成项目背景调研、传统停车场监测方案缺陷分析，确定技术路线；
2. 完成基于 UWB 的无人机相对定位算法设计、扩展卡尔曼滤波（EKF）算法仿真；
3. 搭建 Crazyflie 2.1 无人机硬件平台，完成传感器模块选型与驱动适配；
4. 构建双分支车位识别模型，完成数据集重标注、模型训练与性能验证；
5.完成核心算法仿真、硬件平台调试、模型训练，后续推进算法部署与系统联调。









1. 定位模块：完成基于 UWB 的无人机相对定位方案设计，实现 Token Ring 轮询通信机制，完成扩展卡尔曼滤波（EKF）算法仿真；
2. 硬件平台：完成 3 台 Crazyflie 2.1 四旋翼无人机搭建，适配 Flow Deck v2、UWB 定位甲板、IMU 等传感器模块，搭建 Ubuntu 22.04+ROS2+cfclient 开发环境，实现无人机正常起飞控制；
3. 车位识别：设计双分支跨模态融合车位识别模型，完成 VisDrone 数据集重标注，在 Tesla V100 服务器完成 50 轮模型训练，损失函数收敛，红外与可见光模态车位检测效果良好；










主要问题：
1. 无人机 UWB 相对定位算法实际部署精度不足，集群编队飞行稳定性待提升；
2. 停车场专用双光数据集规模有限，模型泛化能力与实时推理速度需优化；
3. 强化学习集群调度算法未实现硬件部署与效率测试；
4. 无人机机载算力有限，实时数据采集与模型推理难以同步高效运行。
解决方案：
1. 优化 EKF 算法参数，增加无人机通信稳定性调试，逐步实现稳定集群编队；
2. 扩充停车场场景数据集，对比多种检测模型，精简模型结构提升实时推理速度；
3. 完成调度算法仿真验证，优化算法适配嵌入式平台，部署至 Crazyflie 测试采集效率；
4. 适配 AI deck 扩展模块，分担机载算力压力，实现数据采集与推理协同运行。
